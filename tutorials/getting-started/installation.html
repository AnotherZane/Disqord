Installation

This tutorial covers the installation of your IDE and the library. If you have an IDE already setup, skip to NuGet.

IDE

It is a must, for working with a library of this size, to have a good IDE on your hands.

We recommend:

  • Visual Studio 2019 or later:
    • Windows-only
    • has a free Community edition
  • Rider:
    • cross-platform
    • requires a license
Don't feel forced to use just these two! It's okay to use whatever you're more comfortable with, even the .NET CLI alone.
Workload

Depending on the IDE you chose, the workload installation will differ slightly, but should be more or less the same. You will need to install the .NET desktop development workload which includes the latest .NET SDK.

For Visual Studio 2019, you must select the following in the installer:

Solution and Project

Create a console application project.
