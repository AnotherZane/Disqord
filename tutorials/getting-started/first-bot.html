Discord Bot Application

To use the Discord API you'll need to create a Discord bot application first.

You can skip this tutorial if you've made a bot application already (optionally take a look at privileged intents).

Creating the Application

Head over to the Discord Developer Portal and log in. You should be looking at the following screen:

From here, click on the New Application button and give your application a name.

You should be looking at the following screen now:

You can change your application's data here. Here's MyBot with an avatar and a description.

Creating the Bot

The application alone isn't quite enough. Now you'll need to head over to the Bot tab:

Click on Add Bot and confirm it. You should see your bot account created:

Privileged Intents

Last piece of configuration is the privileged intents section. This is a bit of an advanced topic, but to summarize - if your bot is growing, you have to apply for privileged intents that are tied to data that Discord deems sensitive. For now, just toggle on the members intent as it's mandatory for many bot features. Disqord uses this intent by default. You can come back to this later at any time.

Inviting the Bot

To get the bot in your guild, you'll need to invite it. Head over to the OAuth2 tab, scroll down to OAuth2 URL Generator, tick on the bot scope, and copy the generated link.

Open the generated link (paste it into your browser). Select which server to add your bot to. Here MyBot is getting added to the Testing server:

Click Authorize, fill out any captcha, and you should see your bot appear in the specified server.

Visual Studio should now open your newly created solution with the most basic "hello world" code:

Getting the Bot Token

You're all done with the bot application setup. You can now head back to the Bot tab and copy the bot's token. It will be used for authorizing all bot requests and connections.

Your bot's token is like a password. Do not share it with anyone else.
